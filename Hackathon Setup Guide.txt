VS Code
	1. Download/install from https://code.visualstudio.com/
Python
	1. Download/install from https://www.python.org/downloads/

Metamask
	1. Add to browser from https://metamask.io/
Git
	1. Download/install from https://git-scm.com/
	2. You'll probably want everything it offers - at the very least get Git Bash and Git for Windows
